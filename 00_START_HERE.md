## üìä PROJECT OVERVIEW

### What You're Building
Transform an NFT dashboard into a comprehensive analytics platform for the DementiaMithura mobile app.

### 8 Functional Pages
1. **Overview Dashboard** - Key metrics, charts, quick stats
2. **User Analytics** - Demographics, user table, segmentation
3. **Engagement Analytics** - Feature usage, session data
4. **Health Insights** - Symptoms, mood patterns (anonymized)
5. **Content Analytics** - Article performance, gamification
6. **Reminder Analytics** - Medication, voice reminders
7. **Reports & Export** - Generate reports, export data
8. **Settings** - Dashboard config, user management

### Tech Stack
- **Frontend:** Next.js 16, TypeScript, Tailwind CSS v4
- **State:** Zustand (auth, theme, notifications, settings)
- **Data:** React Query (fetching, caching, real-time)
- **Charts:** Recharts (interactive, responsive)
- **Backend:** Next.js API routes or Node.js/Express
- **Database:** PostgreSQL or MongoDB (for analytics)
